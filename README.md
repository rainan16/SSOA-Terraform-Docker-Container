# SSOA-Terraform-Docker-Container

## Prerequisites to run GITHUB Actions

In github, GITHUB_SECRECTS must be set to run the Actions:

```DOCKER_HUB_USERNAME```

```DOCKER_HUB_PASSWORD```

Update [.env](.env) and [.env.test](.env.test) the Dockerhub repository:

```DOCKERHUBNAME```
